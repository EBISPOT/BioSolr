Solr Ontology Update Processor
==============================

Version 0.1
-----------

- Support for adding ontology data to annotated documents, read from an OWL file (either local or remote).


Version 0.2
-----------

- Add support for adding ontology data from the EBI's Ontology Lookup Service beta site.


Version 0.3
-----------

- Add support for reading ontology data from the OLS without knowing the ontology.


Version 0.4
-----------

- Use common ontology-annotator-core library, removing core classes from this module.


Version 0.5
-----------

- Various updates related to Core library, making building OntologyHelper more generic.
- Add optional parent paths to generated data.
